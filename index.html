Wine Recommender System Using Unsupervised Content-Based Filtering

This project aims to created a sophisticated recommendation system designed to offer personalized wine suggestions to users based on their individual taste preferences. The project employs unsupervised machine learning concepts, namely content-based filtering, to perform accurate user-item predictions. The system leverages a comprehensive dataset of wine reviews and employs natural language processing (NLP) techniques to extract qualitative features from wine descriptions, creating detailed taste profiles. The heart of the system lies in neural network models that generate user and item vectors, facilitating accurate predictions of user ratings. Users can interact with the system by providing taste preferences, enabling the recommender to deliver tailored wine recommendations.

Regression and Classification of Professional Soccer Players

This project delves into the application of data mining techniques to extract valuable non-trivial insights from a dataset comprising FIFA-registered professional soccer players. It focuses on two primary objectives: (1) categorizing players based on their positions and (2) predicting the market value of each player.

For the classification tasks, the project employs four distinct techniques: Decision Trees, Random Forest Classification, NaÃ¯ve Bayes, and k-Nearest Neighbors. The primary evaluation metric for these algorithms is accuracy. Additionally, clustering techniques are introduced in an experimental context to assess their potential impact on performance outcomes.

In the realm of prediction tasks, the project adopts multiple linear regression and assesses its performance using metrics such as Root Mean Square Error (RSME) and R2-score. The overarching aim of this project is to gain insights into whether players (a) are positioned correctly and (b) are accurately valued in the transfer market.
